## User Start Guide

1. Open Clarity in your browser (your deployed URL or http://localhost:8080).
2. Enter your API key (OpenAI, Groq, or AssemblyAI) when prompted.
3. Record audio or upload a file.
4. View, edit, and refine transcripts instantly.
5. Export or import your data anytime from Settings.

## Run with Docker

Start instantly with:
```bash
docker run -d -p 8080:8080 registry.ctl.qzz.io/clarity:latest
```
Then open http://localhost:8080 in your browser.
